    3. Plant your own Project

    Start a new project from the boilerplate by copying its contents into a new folder. 

    cd ~/lycheeJS/projects;
cp -R ./boilerplate ./myproject; # Replace myproject with unique identifier

    Modify the unique identifier of the lychee.Environment in the index.html, so you have sweet remote debugging features. 

    var environment = new lychee.Environment({
	// /projects/myproject/source/index.html#L48
	id: 'boilerplate', // Replace boilerplate with unique identifier
});

    Modify the lychee.net.Client's settings, so that it can connect to the lychee.net.Server.

    The API call can be replaced by either a Settings Object (e.g. { host: 'myproject.lycheejs.org', port: 1337 }) or an API call String (e.g. "/api/Server/?identifier=myproject"). 

    var settings = {
	// /projects/myproject/source/Main.js#L16
	client: '/api/Server/?identifier=boilerplate', // Replace boilerplate with unique identifier
};
